Electromagnetic Metameterial-Inspired Devices

These are some of my projects utilizing metamaterial properties.

This project was based on a paper by Yuan Dan Dong, Tao Yang, Tatsuo Itoh in IEEE Xplore called "Substrate Integrated Waveguide Loaded by Complementary Split-Ring Resonators and its Applications to Miniaturized Waveguide Filters."

The animation is a carpet plot of the SIW Diplexer showing isolation between ports 2 and 3.

  • Spring 2016

    • Pong FPGA Using VGA Driver

    This project was done in class at Rose-Hulman. We were given a starting point with source code pulled offline. The project was to make the code more portable. The first task of the project was to design a controller to replace the video timer of the original game. I also was able to add different features like sounds for hit,miss,and wall hit. In addition to the sound features, I was able to implement a score board to count how many times the paddle hits the ball. Once I added these features, I wanted to do more with the project. I experimented with adding another paddle and ulitmately turned it into a two player game.
